## 복습 노트: 소수의 거듭제곱 판별 트릭

---

### 문제

> 주어진 수 n이 소수 p의 거듭제곱인지 반복문/재귀 없이 O(1)로 판별하라

---

### 공식

```python
return n > 0 and (p의_최대_거듭제곱) % n == 0
```

---

### 원리

1. **소수 p**는 재료가 자기 자신 하나뿐
2. p를 여러 번 곱한 수(p^k)의 약수는 **오직 p의 거듭제곱뿐**
3. 따라서 p^k % n == 0이면, n은 반드시 p의 거듭제곱

---

### 비유: 순수한 레고 탑

- 빨간 블록(=p)**만**으로 쌓은 탑 = p^k
- 이 탑을 깔끔하게 나눌 수 있는 조각 = 빨간 블록으로만 된 조각
- 다른 색 블록이 섞인 조각으로는 절대 깔끔하게 나눌 수 없음

---

### 적용 예시 (32-bit int 범위)

| 소수 p | 최대 거듭제곱 | 값              | LeetCode |
| ------ | ------------- | --------------- | -------- |
| 2      | 2³⁰          | 1,073,741,824   | #231     |
| 3      | 3¹⁹          | 1,162,261,467   | #326     |
| 5      | 5¹³          | 1,220,703,125   | -        |
| 7      | 7¹¹          | 1,977,326,743   | -        |

```python
# Power of Two
def is_power_of_two(n: int) -> bool:
    return n > 0 and 1073741824 % n == 0

# Power of Three
def is_power_of_three(n: int) -> bool:
    return n > 0 and 1162261467 % n == 0
```

---

### 주의: 합성수에는 적용 불가

**4 (= 2×2)**의 경우:

```
4³ = 64
64 % 8 == 0 → True  (하지만 8은 4의 거듭제곱이 아님!)
```

합성수는 재료가 여러 개 섞여있어서, 거듭제곱이 아닌 약수가 끼어듦
