## 1. 바리스타 업무 실태

**조사 에이전트:** 카페-현장-조사원

### 1.1 매일 아침 오픈 루틴 (10-15분 소요)

- 그룹헤드 플러싱으로 잔여물 제거
- 수압 및 온도 안정성 확인
- 테스트 샷 추출 → 맛보기 → 조정 반복
- 원두 에이징 상태와 환경 습도에 맞춘 그라인더 분쇄도 조정
- 각 쉬프트별 원두 계량, 하우스 블렌드/디카페인/스페셜티 정리 및 라벨링
- 피크 시간대 대비 2-3개 호퍼 사전 세팅
- 하루 중 2-3회 추가 미세 조정 필요

### 1.2 원두 로스팅 후 경과일에 따른 추출 변화

> **팩트체크: 실제로 존재하는 현상**

| 경과일 | 상태 |
|--------|------|
| 로스팅 직후~5일 | 떫고 플랫한 맛 (CO2 과다) |
| 7-8일 | 플레이버 발현 시작 (라벤더, 유자 등) |
| 12-15일 | 플레이버, 단맛, 마우스필 정점 |
| 15일 이후 | 점진적 품질 저하 |

- 디개싱 기간: 보통 2-14일 필요
- 다크 로스트일수록 디개싱이 빠름

### 1.3 날씨/습도에 따른 그라인더 조정

> **팩트체크: 필수적인 작업**

| 날씨 | 영향 | 대응 |
|------|------|------|
| 습한 날 | 원두가 습기 흡수 → 뭉침 → 추출 느려짐 | 더 굵게 분쇄 |
| 건조한 날 | 빨리 추출됨 | 더 가늘게 분쇄 |

- 추출 실패율 약 **15%**
- 주된 원인: 분쇄도와 탬핑 불균형

## 2. 카페 사장님 실제 고충

**조사 에이전트:** 카페-현장-조사원 + 시장-회의론자

### 2.1 한국 카페 시장 현황

| 지표 | 수치 |
|------|------|
| 전국 커피전문점 수 | 약 **10만 개** (2024년 기준, 편의점보다 많음) |
| 브랜드 수 | **886개** |
| 서울 카페 1년 생존율 | **77.7%** (4곳 중 1곳 1년 내 폐업) |
| 2024년 카페 폐업 수 | 약 **12,242개** (하루 평균 34곳) |
| 2025년 1분기 서울시 커피/음료업 | 폐업률(4.0%)이 개업률(3.1%)을 **10년 만에 처음 역전** |
| 2030세대 커피전문점 | 창업 대비 폐업률 **82.2%** |

### 2.2 카페 사장님 Top 5 페인포인트

1. **생존 위기 + 저가 프랜차이즈 가격 경쟁**
   - 메가커피, 컴포즈 등 1,500원대 가격

2. **원재료비 급등**
   - 아라비카 원두 가격 전년 대비 60%+ 급등
   - 톤당 $5,158 → $8,300

3. **높은 고정비 (임대료 + 인건비)**
   - 부부 운영 시 순익: 월 300-400만원
   - 개인 카페 월매출: 400-600만원이 대부분
   - 순이익률: 15-25%
   - 월 순수익: 100-200만원이 현실

4. **고객 유치/마케팅 부담**
   - 네이버 플레이스, 인스타 릴스, 블로그 등 필수

5. **재고/품질 관리 복잡성**

## 3. 프랜차이즈 품질 관리 실태

**조사 에이전트:** 카페-현장-조사원

### 3.1 현재 품질 관리 방식

| 방식 | 상세 |
|------|------|
| 슈퍼바이저(SV) | "프랜차이즈의 꽃", SV 1인당 10-15개 매장 관리 추정 |
| 미스터리 쇼퍼 | 건당 10-15만원, 월 1-2회 |
| 체크 항목 | 주문 말투, 결제 과정, 음료 나오는 속도, 컵의 얼음 개수까지 |
| 컴포즈커피 | 자체 로스팅 공장 운영, 수석 로스터가 직접 생두 선별/로스팅, 전국 가맹점 직접 공급 |

### 3.2 매장 간 품질 편차

> **팩트체크: 실제로 큰 문제**

- **빽다방:** 성수점 1.6 TDS vs 연희점 1.2 TDS (같은 브랜드인데 농도 차이)
- **메가커피:** 138건 식품위생법 위반
- **컴포즈커피:** 135건 식품위생법 위반
  - 위생교육 미이수: 45.3%
  - 기준/규격 위반: 23.8%
- **주요 원인:** 전자동 머신 관리 문제 + 교육 미흡 직원

## 5. 솔루션 평가

**조사 에이전트:** 솔루션-평가자

### 5.1 "긍정만" 시스템 리스크

| 지표 | 수치 |
|------|------|
| 부정적 리뷰를 의도적으로 탐색하는 소비자 | **96%** (PowerReviews) |
| 1점 리뷰를 특별히 찾는 소비자 | **52%** (밀레니얼 60%) |
| 5점 만점 평점에 의심하는 소비자 | **46%** (Z세대 53%) |
| 온라인 리뷰 신뢰도 | 2020년 80% → 2025년 **42%**로 급락 |

### 5.2 게이미피케이션 지속성 데이터

| 지표 | 수치 |
|------|------|
| 일반 앱 30일 리텐션율 | **4-10%** |
| 에페메럴 콘텐츠 FOMO 효과 | 68%가 사라지는 콘텐츠에 더 적극 반응 |
| 게이미피케이션 도입 시 고객 참여도 | 최대 **48% 상승** (단기) |
| 장기 참여 시 | "평판 유지에 대한 심리적 스트레스" → 피로도 발생 |

### 5.3 푸시 알림 피로도

| 지표 | 수치 |
|------|------|
| 평균 모바일 사용자 하루 푸시 수신 | **46개** |
| 주 1개 푸시만으로 알림 끄는 비율 | **10%** |
| 주 3-6개 푸시 시 알림 비활성화 | **40%** |
| 알림 스팸 이유 - 너무 잦음 | 62% |
| 알림 스팸 이유 - 관련 없음 | 55% |
| 푸시에 짜증을 느끼는 비율 | **49%** |
| 과도한 알림 → 앱 삭제 | 직결 |

### 5.4 QR/PWA 접근성

- QR 코드: 레스토랑 70%+ 이미 사용, 78% 소비자 편의성 선호
- **콜드스타트 이론** (Andrew Chen): 좁은 지역 집중 → 유동성 달성 후 확장

### 5.5 Year 1 MAU 15,000 목표 역산

| 단계 | 수치 |
|------|------|
| 성수동 일일 유동인구 | 약 65,500명 |
| 카페 방문 비율 (~30%) | 약 19,650명/일 |
| 앱 다운로드 전환율 (1-3%) | 월 200-600명 |
| 30일 리텐션 (4-10%) | 극소수 유지 |
| 월 리텐션 20% (매우 높은 가정) 시 | 누적 7.5만 다운로드 필요 |
| 월 리텐션 10% (현실적) 시 | 누적 **15만 다운로드** 필요 |

> **결론:** 성수동만으로는 MAU 15,000 달성 불가

## 6. 종합 팩트체크 결론

### 6.1 종합 점수

| 항목 | 점수 |
|------|------|
| 문제 정의 부합도 | 5.0 / 10 |
| 솔루션 실현 가능성 | 5.0 / 10 |
| 경쟁 우위 지속성 | 4.0 / 10 |
| **종합** | **4.7 / 10** |

### 6.2 4명 전원 합의 사항

1. **프랜차이즈 품질관리 B2B**가 가장 현실적인 PMF (Product-Market Fit)
2. 개인 카페 B2B 유료 전환은 어려움 (순수익 대비 과도한 가격 + 무료 대안 다수)
3. "커피 맛이 매일 다르다"는 기술적 사실이나, **소비자 인식은 제한적**
4. **콜드스타트 문제**가 심각 (24시간 소멸이 악화)
5. **네이버 플레이스**가 가장 큰 경쟁 위협

### 6.3 핵심 6개 권고

| # | 권고 사항 |
|---|-----------|
| 1 | 사업 축을 **개인 카페 B2B → 프랜차이즈 Enterprise B2B**로 이동 |
| 2 | 24시간 완전 소멸 → **시간 가중치 시스템**으로 전환 |
| 3 | 3단계 반응에 **맛 태그 추가**로 데이터 해상도 확보 |
| 4 | Best Ever 해금 조건 대폭 완화 (30+10 → **10+3**) |
| 5 | "오늘의 맛" → **"내 취향에 맞는 카페 발견"**으로 문제 재정의 검토 |
| 6 | **"긍정만" 정책 재검토** (건설적 피드백 허용) |
