# n & (n - 1) == 0 으로 2의 거듭제곱 판별하기

## 전등 스위치로 이해하기

이진수의 각 자리를 **전등 스위치**라고 상상해보자. 1이면 켜짐, 0이면 꺼짐.

**2의 거듭제곱은 전등이 딱 1개만 켜져 있다:**

```
1  = 0001  → 🔴⚫⚫⚫  (전등 1개)
2  = 0010  → ⚫🔴⚫⚫  (전등 1개)
4  = 0100  → ⚫⚫🔴⚫  (전등 1개)
8  = 1000  → ⚫⚫⚫🔴  (전등 1개)
```

**2의 거듭제곱이 아닌 수는 전등이 2개 이상:**

```
3  = 0011  → 🔴🔴⚫⚫  (전등 2개)
6  = 0110  → ⚫🔴🔴⚫  (전등 2개)
```

---

## n - 1을 하면 무슨 일이 일어날까?

### 전등이 1개인 경우 (2의 거듭제곱)

```
8   = 1000  → ⚫⚫⚫🔴
8-1 = 0111  → 🔴🔴🔴⚫
```

유일하게 켜져있던 전등이 꺼지고, 그 아래 전등이 전부 켜진다.

마치 **1000원짜리 지폐를 동전으로 바꾸면 500 + 100 + ... 으로 쪼개지는 것**과 같다.

결과적으로 `n`과 `n-1`이 **겹치는 자리가 하나도 없으므로** `&` 결과가 0이다:

```
  1000   (n)
& 0111   (n-1)
------
  0000   ✓ 완전히 반대!
```

### 전등이 2개 이상인 경우 (2의 거듭제곱이 아님)

```
6   = 0110  → ⚫🔴🔴⚫
6-1 = 0101  → 🔴⚫🔴⚫
```

가장 오른쪽 전등 하나만 바뀌고, **왼쪽의 다른 전등은 그대로 켜져 있다:**

```
  0110   (n)
& 0101   (n-1)
------
  0100   ✗ 0이 아님! (겹치는 전등이 남아있음)
```

## 핵심 정리

| 구분                        | n      | n - 1  | 겹침               |
| --------------------------- | ------ | ------ | ------------------ |
| 전등 1개 (2의 거듭제곱)     | `1000` | `0111` | 없음 → **0**       |
| 전등 2개 이상               | `0110` | `0101` | 있음 → **0 아님**  |

> **전등이 딱 1개면**, 1을 빼는 순간 그 전등이 꺼지고 아래가 다 켜지니까 겹칠 수가 없다.
> **전등이 2개 이상이면**, 1을 빼도 위쪽 전등이 살아남아서 반드시 겹친다.

그래서 `n & (n - 1) == 0`이면 전등이 1개 = 2의 거듭제곱이다.

## 응용: 4의 거듭제곱 판별

4의 거듭제곱은 2의 거듭제곱이면서, 1이 **짝수 번째 비트 위치**(0, 2, 4, 6번째)에 있다:

```
1   = 0b0000 0001   (0번째 비트) ✓
4   = 0b0000 0100   (2번째 비트) ✓
16  = 0b0001 0000   (4번째 비트) ✓
64  = 0b0100 0000   (6번째 비트) ✓

2   = 0b0000 0010   (1번째 비트) ✗
8   = 0b0000 1000   (3번째 비트) ✗
```

짝수 자리만 1인 마스크 `0x55555555 = 0b0101...0101`과 AND 연산으로 구분:

```python
def is_power_of_four(n: int) -> bool:
    return n > 0 and n & (n - 1) == 0 and n & 0x55555555 != 0
```

또는 수학적으로, `4^k = (3+1)^k`이므로 3으로 나눈 나머지가 항상 1:

```python
def is_power_of_four(n: int) -> bool:
    return n > 0 and n & (n - 1) == 0 and n % 3 == 1
```
