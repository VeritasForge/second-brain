# 326. Power of Three — `(3**31) % n == 0` 이 왜 동작하는가?

## 배경 지식: 소인수(Prime Factor)란?

**소수(Prime)**: 1과 자기 자신으로만 나누어지는 수 (2, 3, 5, 7, 11, 13...)
**인수(Factor)**: 어떤 수를 나누어 떨어지게 하는 수
**소인수** = 인수 중에서 소수인 것

어떤 숫자를 **더 이상 쪼갤 수 없는 레고 조각**으로 분해한다고 생각하면:

```
12를 분해해보자:

12 = 2 × 6       ← 6은 더 쪼갤 수 있다
   = 2 × 2 × 3   ← 2와 3은 더 이상 쪼갤 수 없다! (소수니까)
```

여기서 **2**와 **3**이 12의 **소인수**이고, 이 과정을 **소인수분해**라고 합니다.

| 숫자 | 소인수분해      | 소인수  |
| ---- | --------------- | ------- |
| 12   | 2 × 2 × 3      | 2, 3    |
| 27   | 3 × 3 × 3      | 3       |
| 30   | 2 × 3 × 5      | 2, 3, 5 |
| 81   | 3 × 3 × 3 × 3  | 3       |

> **한 줄 요약**: 소인수 = 어떤 수를 끝까지 쪼갰을 때 나오는 소수 조각들

---

## 1단계: 코드가 하는 일

```python
def is_power_of_three(n: int) -> bool:
    return n > 0 and (3**31) % n == 0
```

3^31은 충분히 큰 3의 거듭제곱입니다. Python은 정수 크기 제한이 없으므로 얼마든지 큰 값을 쓸 수 있습니다.

> **참고**: 문제의 제약 조건 `-2^31 <= n <= 2^31 - 1`은 **n의 범위**가 32비트 signed int라는 뜻입니다. 32비트 signed int에서 양수 최대값은 `2^31 - 1 = 2,147,483,647`이고, 이 범위 내 가장 큰 3의 거듭제곱은 **3^19 = 1,162,261,467**입니다 (3^20 = 3,486,784,401은 이미 초과). 따라서 `3**19`를 써도 되고, `3**31`처럼 더 큰 값을 써도 결과는 동일합니다 — 3^31의 약수 중 n 범위에 들어오는 것이 3^0 ~ 3^19뿐이기 때문입니다.

---

## 2단계: 3^31의 약수는 뭘까?

3은 **소수**이기 때문에, 3^31을 소인수분해하면 `3 × 3 × 3 × ... × 3` (31개)밖에 안 나옵니다. 다른 소인수가 없습니다.

따라서 3^31의 약수는 **오직 3의 거듭제곱뿐**입니다:

```
3^31의 약수 = {3^0, 3^1, 3^2, 3^3, ..., 3^31}
            = {1, 3, 9, 27, 81, ...}
```

---

## 3단계: `3^31 % n == 0`의 의미

이건 "3^31이 n으로 나누어 떨어지는가?" = "**n이 3^31의 약수인가?**"와 같은 말입니다.

- **n = 9 (= 3^2)** → 3^31 ÷ 3^2 = 3^29 → 나머지 0 → `True`
- **n = 12 (= 3 × 4)** → 소인수에 2가 포함 → 3^31 ÷ 12 → 나누어 떨어지지 않음 → `False`
- **n = 6 (= 2 × 3)** → 소인수에 2가 포함 → 3^31에는 소인수 2가 없으므로 → 나누어 떨어지지 않음 → `False`

## 현실 비유: 레고 탑

- 3^31은 **빨간 블록 31개**로만 쌓은 탑이에요
- "이 탑을 n개씩 똑같이 나눌 수 있나요?"라고 물어보는 거예요
- n이 빨간 블록으로만 구성되어 있으면 (3의 거듭제곱) → 깔끔하게 나눠져요
- n에 파란 블록(다른 소인수)이 하나라도 섞여 있으면 → 절대 나눠질 수 없어요

---

## 왜 3이라서 가능한가?

**3이 소수이기 때문**입니다. 만약 같은 방법을 **6의 거듭제곱 판별**에 쓰면 안 됩니다:

```
6^10의 약수에는 6의 거듭제곱이 아닌 것도 포함됨
예: 12는 6^10의 약수이지만, 6의 거듭제곱이 아님 (6^1=6, 6^2=36)
```

6 = 2 × 3이라 소인수가 여러 개이므로, 약수 중에 6의 거듭제곱이 아닌 것(예: 2, 4, 12...)이 섞여 들어옵니다.

---

## 정리

| 조건             | 이유                                              |
| ---------------- | ------------------------------------------------- |
| `n > 0`          | 0이나 음수는 거듭제곱이 될 수 없으므로            |
| `3**31 % n == 0` | 3이 소수이므로, 3^31의 약수 = 3의 거듭제곱만 존재 |

**핵심 공식**: 소수 p에 대해, p^k의 약수는 반드시 p의 거듭제곱이다. 이 성질 덕분에 이 트릭이 성립합니다.
